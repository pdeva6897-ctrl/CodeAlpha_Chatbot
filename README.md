# 🤖 Simple Chatbot

A simple rule-based chatbot developed using Python.

This project was created as part of the CodeAlpha Internship Program. The chatbot interacts with users through predefined responses based on user input.

---

## ✨ Features

- Simple console-based chatbot
- Predefined responses
- Interactive conversation
- Beginner-friendly Python project
- Easy to understand and modify

---

## 🛠️ Technologies Used

- Python
